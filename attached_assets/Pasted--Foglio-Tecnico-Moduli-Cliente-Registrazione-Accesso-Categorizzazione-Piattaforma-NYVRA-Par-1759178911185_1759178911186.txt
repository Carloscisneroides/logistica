🔐 Modulo di Registrazione Cliente
Input richiesto
Email (validazione formale + DNS)

Password (min. 12 caratteri, entropia controllata)

Nome e Cognome (opzionale)

Lingua preferita

Accettazione Termini & Privacy

Logica antifrode
Rate limiting per IP e device fingerprint

Verifica email via OTP

Logging tentativi falliti

Output
Creazione profilo con UUID

Flag email_verified: false

Token sessione temporaneo

✅ Modulo di Conferma Accesso (Login)
Input
Email + Password

Sicurezza
Protezione brute-force (CAPTCHA dopo 5 tentativi)

Device fingerprinting + geolocalizzazione IP

Logging accessi con timestamp e user-agent

Output
Verifica hash password

Se email non verificata → re-invio OTP

Se OK → generazione JWT/session token

📲 Conferma Accesso In-App
Flusso mobile-native
Login → OTP → Dashboard

OTP via push/SMS

Token salvato in Keychain/Keystore

UI multilingua, animazioni leggere
